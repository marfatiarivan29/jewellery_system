# Jewellery System

A comprehensive jewellery inventory and sales management system built with Laravel.

## Features (Planned)
- Inventory Management
- Sales Tracking
- Customer Profiles
- Reporting and Analytics

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/marfatiarivan29/jewellery_system.git
   ```
2. Install PHP dependencies:
   ```bash
   composer install
   ```
3. Install NPM dependencies:
   ```bash
   npm install && npm run build
   ```
4. Configure environment:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```
5. Run migrations:
   ```bash
   php artisan migrate
   ```
